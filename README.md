# nondeterminism.io

Canonical notes on nondeterminism in agentic systems. Definitions, failure modes, and replay-first primitives for reasoning about agent behavior in production.

## Structure

- `index.html` - Landing page
- `essay.html` - Core essay on why nondeterminism breaks agentic systems
- `taxonomy.html` - Reference documentation of nondeterminism sources
- `styles.css` - Minimal styling

## Design principles

- Plain HTML + CSS only
- No JavaScript, no build step, no dependencies
- RFC / design-doc aesthetic
- System fonts, generous whitespace
- Maximum readability
